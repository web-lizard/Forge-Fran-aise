# Forge Française

Имперский мозговыбиватель французского языка.

Mobile-first учебный движок французского языка на Vue 3, FastAPI и JSON-контенте.

## Ports

Backend:
http://127.0.0.1:8797/api/health

Frontend:
http://127.0.0.1:5197
